## What You’ll Need to Create Your Infographic

You’ll need access to both **Adobe InDesign** and **Adobe Illustrator**, and to download the following images and fonts from Envato Elements:

- [Space Backgrounds](https://elements.envato.com/space-backgrounds-YVR2HR)
- [Space Flat Icon Set](https://elements.envato.com/space-modern-flat-design-icon-set-AUXM5V) (Some free icons can be downloaded **[here](https://image-control-storage.s3.amazonaws.com/space-icons.zip)**[)](https://image-control-storage.s3.amazonaws.com/space-icons.zip)
- Flat illustration of a Space Shuttle (A free version can be downloaded **[here](https://image-control-storage.s3.amazonaws.com/space_shuttle_the_launch_to_the_sky.zip)**)
- [BW Modelica](https://elements.envato.com/bw-modelica-font-family-96W3JH) (find a free replacement on google fonts)

## 1. How to Create a Web Document in InDesign

### Step 1

Open **InDesign** and go to **File &gt; New &gt; Document.**

With the intent set to **Web**, set the **Width** of the page to **435 px** and **Height** to **1300 px.** This is a good size for sharing on sites like Pinterest.

Set the **Margins** to **36 px**, and click **Create**.

<div class="wp-block-image"><figure class="aligncenter">![new document](https://image-control-storage.s3.amazonaws.com/2020/03/30095621/1.1-5.jpg)</figure></div>### Step 2

Expand the **Layers** panel **(Window &gt; Layers)**, and double-click on ***Layer 1***, renaming it ***Background***.

Create four more new layers—***Arrows, Rocket, Icons*** and finally, ***Type***.

Then lock all the layers except ***Background***.

<div class="wp-block-image"><figure class="aligncenter">![type layer](https://image-control-storage.s3.amazonaws.com/2020/03/30095623/1.2-4.jpg)</figure></div>## 2. How to Create a Celestial Backdrop 

### Step 1

Use the **Rectangle Frame Tool (F)** to create an image frame across the whole page.

**File &gt; Place**, navigate to the **Space Backgrounds**<span class="tadv-color" style="color:#cf2e2e"> (Use a free version of our choice. Nasa has a lot of free images you can download)</span> image. Click **Open** and allow it to fill up the whole frame.

<div class="wp-block-image"><figure class="aligncenter">![placed image](https://image-control-storage.s3.amazonaws.com/2020/03/30095625/2.1-4.jpg)</figure></div>### Step 2

Switch to the **Rectangle Tool (M)** and drag across the top of the page to create a rough square shape.

From the **Swatches** panel **(Window &gt; Color &gt; Swatches)**, set the **Fill** to **\[Black\].**

<div class="wp-block-image"><figure class="aligncenter">![black shape](https://image-control-storage.s3.amazonaws.com/2020/03/30095626/2.2-3.jpg)</figure></div>With the shape selected, go to **Object &gt; Effects &gt; Gradient Feather.** Set the **Type** to **Linear** and allow the gradient to extend from darker at the top to transparent towards the bottom of the shape. Click **OK**.

<div class="wp-block-image"><figure class="aligncenter">![gradient feather](https://image-control-storage.s3.amazonaws.com/2020/03/30095628/2.2.1-3.jpg)</figure></div>### Step 3

Select the shape and **Edit &gt; Copy, Edit &gt; Paste** it. Then, on the copy, **Right-Click &gt; Transform &gt; Flip Vertical.**

<div class="wp-block-image"><figure class="aligncenter">![flip vertical](https://image-control-storage.s3.amazonaws.com/2020/03/30095629/2.3-3.jpg)</figure></div>Move to the bottom of the page, and extend the height of the shape upwards to about halfway up the page.

<div class="wp-block-image"><figure class="aligncenter">![background layer](https://image-control-storage.s3.amazonaws.com/2020/03/30095631/2.3.1-3.jpg)</figure></div>### Step 4

Open the [**space shuttle** ](https://image-control-storage.s3.amazonaws.com/space_shuttle_the_launch_to_the_sky.zip)illustration in **Illustrator**.

Select only the rocket shape, plus one of the transparent layers of smoke along the bottom of the image, pictured below. Then **Copy** these.

<div class="wp-block-image"><figure class="aligncenter">![rocket graphic](https://image-control-storage.s3.amazonaws.com/2020/03/30095632/2.4-3.jpg)</figure></div>Create a new Illustrator document, and **Paste** these elements onto the page, before going to **File &gt; Save As** and saving this as an Illustrator EPS file.

<figure class="wp-block-image">![rocket graphic](https://image-control-storage.s3.amazonaws.com/2020/03/30095634/2.4.1-3.jpg)</figure>### Step 5

Return to your **InDesign** document and lock the ***Background*** layer. Unlock the ***Rocket*** layer.

Create an image frame using the **Rectangle Frame Tool (F)** at the bottom of the page and **File &gt; Place**, choosing the edited rocket EPS image. **Open**, and make sure the rocket image is centered on the page.

<figure class="wp-block-image">![rocket image](https://image-control-storage.s3.amazonaws.com/2020/03/30095635/2.5-4.jpg)</figure>## 3. How to Create a Starry Header for Your Infographic

### Step 1

Lock the ***Rocket*** layer and unlock the top layer, ***Type***.

Zoom in on the top of the page, take the **Type Tool (T)**, and create a text frame across the top-center of the page.

Type in ‘SPACE’, and from either the top **Controls** panel or the **Character** and **Paragraph** panels **(Window &gt; Type &amp; Tables &gt; Character/Paragraph)**, set the **Font** to **BW Modelica Black, Size 100 pt, Tracking 100**, and **Align Center.**

From the **Swatches** panel, set the **Font Color** to **\[Paper\].**

<div class="wp-block-image"><figure class="aligncenter">![character panel](https://image-control-storage.s3.amazonaws.com/2020/03/30095637/3.1-6.jpg)</figure></div>### Step 2

Add two text frames above and below ‘SPACE’, set in **BW Modelica ExtraBold, Size 30 pt, Tracking 230**, and **Align Center.**

The first should read ‘THE FUTURE OF’ and the bottom simply ‘TRAVEL’.

<div class="wp-block-image"><figure class="aligncenter">![text frame](https://image-control-storage.s3.amazonaws.com/2020/03/30095639/3.2-4.jpg)</figure></div>### Step 3

Select the ‘SPACE’ text frame and go to Type &gt; Create Outlines to vectorize the text.

<div class="wp-block-image"><figure class="aligncenter">![create outlines](https://image-control-storage.s3.amazonaws.com/2020/03/30095641/3.3-3.jpg)</figure></div>Set the **Color Fill** of the outlined text to **\[None\].**

<div class="wp-block-image"><figure class="aligncenter">![outlined text](https://image-control-storage.s3.amazonaws.com/2020/03/30095642/3.3.1-3.jpg)</figure></div>Then, go to File &gt; Place, choose an image from the Space Backgrounds folder (I’ve chosen ‘04.jpg’ for contrast), and Open it, allowing it to fill the whole word.

<div class="wp-block-image"><figure class="aligncenter">![space image](https://image-control-storage.s3.amazonaws.com/2020/03/30095643/3.3.2-6.jpg)</figure></div>## 4. How to Build Up Graphics on Your Layout

### Step 1

Lock the ***Type*** layer and unlock the ***Arrows*** layer.

Use the **Line Tool ()** to create a vertical line running from the top of the rocket to about a quarter of the way down the page.

Set the **Stroke Color** to **\[Paper\]**. From the **Stroke** panel **(Window &gt; Stroke)**, set the **Weight** of the line to **20 pt**, and add a **Triangle** shape to the **End** of the line.

<div class="wp-block-image"><figure class="aligncenter">![arrow stroke](https://image-control-storage.s3.amazonaws.com/2020/03/30095647/4.1-3.jpg)</figure></div>### Step 2

With the line selected, go to **Object &gt; Effects &gt; Gradient Feather.**

Add a **Linear** gradient at a **90 Degree Angle**, allowing the arrow to be more transparent towards the base.

<div class="wp-block-image"><figure class="aligncenter">![gradient feather](https://image-control-storage.s3.amazonaws.com/2020/03/30095648/4.2-3.jpg)</figure></div>### Step 3

**Please copy** and **Paste** the arrow, moving it over to the right side of the page, and shortening it so that it is positioned about halfway up the page.

<div class="wp-block-image"><figure class="aligncenter">![pasted arrow](https://image-control-storage.s3.amazonaws.com/2020/03/30095650/4.3-3.jpg)</figure></div>**Paste** more arrows across the page, in the formation shown below, shortening them as needed.

<div class="wp-block-image"><figure class="aligncenter">![arrows pasted](https://image-control-storage.s3.amazonaws.com/2020/03/30095651/4.3.1-11.jpg)</figure></div>### Step 4

Open up the [**space icon set**](https://image-control-storage.s3.amazonaws.com/space-icons.zip) in **Illustrator**.

**Right-Click &gt; Ungroup** the image, so you can select icons individually.

<div class="wp-block-image"><figure class="aligncenter">![ungroup icons](https://image-control-storage.s3.amazonaws.com/2020/03/30095653/4.4-8.jpg)</figure></div>Select the ‘Mars’ icon on the fourth row down, and **Edit &gt; Copy** it.

<div class="wp-block-image"><figure class="aligncenter">![mars icon](https://image-control-storage.s3.amazonaws.com/2020/03/30095654/4.4.1-4.jpg)</figure></div>### Step 5

Back in your **InDesign** document, lock the ***Arrows*** layer and unlock the ***Icons*** layer.

**Edit &gt; Paste** the icon onto the page, and move it so that it sits at the top of the central arrow.

<div class="wp-block-image"><figure class="aligncenter">![pasted icon](https://image-control-storage.s3.amazonaws.com/2020/03/30095655/4.5-4.jpg)</figure></div>From the **Swatches** panel, choose **New Color Swatch** from the panel’s main menu. Create a new RGB swatch, **R=41 G=156 B=186**. Click **Add** and **Done**.

<div class="wp-block-image"><figure class="aligncenter">![new rgb swatch](https://image-control-storage.s3.amazonaws.com/2020/03/30095656/4.5.1-4.jpg)</figure></div>Apply this swatch to the **Fill** of the circle sitting behind the planet.

<div class="wp-block-image"><figure class="aligncenter">![blue fill](https://image-control-storage.s3.amazonaws.com/2020/03/30095657/4.5.2-4.jpg)</figure></div>### Step 6

Head back to the Illustrator icons and select the ‘Rocket’ icon. **Copy** it.

<div class="wp-block-image"><figure class="aligncenter">![rocket icon](https://image-control-storage.s3.amazonaws.com/2020/03/30095658/4.6-2.jpg)</figure></div>**Paste** this directly onto your InDesign document, and position it at the top of the arrow on the far-right side of the page.

Create a new RGB swatch, **R=239 G=79 B=77**, and apply this to the circle. **Right-Click &gt; Transform &gt; Rotate** the rocket to a **45** degree angle.

<div class="wp-block-image"><figure class="aligncenter">![rocket icon](https://image-control-storage.s3.amazonaws.com/2020/03/30095659/4.6.1-2.jpg)</figure></div>### Step 7

Continue to copy and paste more icons, ‘Astronaut’, ‘Orbit’, and ‘Earth’, onto the infographic, placing these in the positions shown below.

Create new RGB swatches of your own choosing and apply these to each of the background circles.

<div class="wp-block-image"><figure class="aligncenter">![earth icon](https://image-control-storage.s3.amazonaws.com/2020/03/30095700/4.7-2.jpg)</figure></div>### Step 8

Isolate the figure of the ‘Alien’ on the icon image and copy and paste it onto the infographic, too.

<div class="wp-block-image"><figure class="aligncenter">![alien icon](https://image-control-storage.s3.amazonaws.com/2020/03/30095701/4.8-2.jpg)</figure></div>Position it on the bottom-right of the layout.

<div class="wp-block-image"><figure class="aligncenter">![alien graphic](https://image-control-storage.s3.amazonaws.com/2020/03/30095702/4.8.1-2.jpg)</figure></div>### Step 9

Lock the ***Icons*** layer and unlock the ***Type*** layer.

With your icons in place, you can now start to build up text on your infographic, to match each icon.

Select the **Type Tool (T)** and create a text frame to the right of the ‘Mars’ icon at the top. Type in a subtitle, and set the **Font** to **BW Modelica ExtraBold, Size 30 pt, Tracking 90**, and **Align Right.** Set the **Font Color** to **\[Paper\].**

<div class="wp-block-image"><figure class="aligncenter">![sub-heading](https://image-control-storage.s3.amazonaws.com/2020/03/30095703/4.9-2.jpg)</figure></div>### Step 10

Create a second text frame below this, typing in your chosen fact, and setting the **Font** to **BW Modelica Bold, Size 12 pt, Tracking 60**, and **Align Right.**

<div class="wp-block-image"><figure class="aligncenter">![body text](https://image-control-storage.s3.amazonaws.com/2020/03/30095704/4.10-2.jpg)</figure></div>### Step 11

Now you can select both the heading and body text frames and **Copy** and **Paste** these, to populate your infographic with more text.

<div class="wp-block-image"><figure class="aligncenter">![text alignment](https://image-control-storage.s3.amazonaws.com/2020/03/30095705/4.11-3.jpg)</figure></div>Adjust the text alignment as you go to suit their position on the page.

<div class="wp-block-image"><figure class="aligncenter">![text frames](https://image-control-storage.s3.amazonaws.com/2020/03/30095706/4.11.1-2.jpg)</figure></div>### Step 12

If you have any areas of the infographic where the text is difficult to read because of the background, you can create circles in **\[Black\]** using the **Ellipse Tool (L).**

<div class="wp-block-image"><figure class="aligncenter">![ellipse tool](https://image-control-storage.s3.amazonaws.com/2020/03/30095708/4.12-3.jpg)</figure></div>Go to **Object &gt; Effects &gt; Gradient Feather** to apply a **Radial** gradient to these shapes.

<div class="wp-block-image"><figure class="aligncenter">![gradient](https://image-control-storage.s3.amazonaws.com/2020/03/30095709/4.12.1-2.jpg)</figure></div>Then make sure to **Right-Click &gt; Arrange &gt; Send to Back** to push them behind the text.

<div class="wp-block-image"><figure class="aligncenter">![send to back](https://image-control-storage.s3.amazonaws.com/2020/03/30095710/4.12.2-2.jpg)</figure></div>### Step 13 

If you want to make a feature of certain statistics, you can set numbers in large-size **BW Modelica.**

<div class="wp-block-image"><figure class="aligncenter">![large type](https://image-control-storage.s3.amazonaws.com/2020/03/30095712/4.13-3.jpg)</figure></div>Then go to **Type &gt; Create Outlines.**

This will really make the figure pop on the page.

<figure class="wp-block-gallery has-nested-images columns-default is-cropped wp-block-gallery-11 is-layout-flex wp-block-gallery-is-layout-flex"><figure class="wp-block-image size-large">![](https://image-control-storage.s3.amazonaws.com/2020/03/30095155/4.13.2.jpg)</figure></figure>## Conclusion: Your Finished Infographic
